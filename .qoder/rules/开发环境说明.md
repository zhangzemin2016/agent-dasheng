# 开发环境说明

基于conda 创建的 langchain环境，所有命令执行都应基于此环境，可以使用`conda activate langchain`激活该环境

## 语言与框架

- python
- langchain
- langGraph
- deep Agents(深度集成langChain 与 langGraph的智能体构建框架)
- flet (用户交互框架)

## 运行环境

使用 conda activate langchain 切换到该环境才可以正常运行

## 开发规范

- 仔细分析需求
- 优先使用主流框架
- 常用方法提取为公共方法
- 要有完善的异常处理
